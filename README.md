# 簡介
實驗室網站，由101學年入學已經畢業的研究生陳威齊製作，使用Ruby on rails開發
> ruby 2.2.2p95
>
> Rails 4.2.5.2

## wiki結構
```
wiki
  ├── Dev_Notes            # 開發紀錄
  ├── MuCat_v1_UI          # 第一版的UI
  ├── UI_note              # UI筆記
  ├── features             # 開發feature的筆記
  ├── icons                # 實驗室網站的icon
  ├── img                  # 筆記會用到的圖片
  ├── meetup               # 與老師討論的紀錄
  └── userstory_versions   # 使用者故事
```
